## COMPONENT/ECOSYSTEMS

- ecosystems ディレクトリには Atomic Components の 「Ecosystems」レベルの component を配置する
- Atomic Design の 「organisms」レベル
- ファイルの接頭語は「Ec」をつける
  - 例：EcForms.tsx
- component の import は atoms, molecules, organisms のみ
- EntryPoint.tsx すべてまとめる
